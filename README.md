\# RPG 126541



Applicativo Java che implementa un Gioco di Ruolo.

Progetto per il corso di Metodologie di Programmazione - AA 2025/26.

Università degli Studi di Camerino.



\---



\## Come eseguire il progetto



\### Prerequisiti

\- Java 25 (LTS)

\- Gradle (incluso nel wrapper)



\### Istruzioni



```bash

git clone https://github.com/LorenzoFino/rpg126541.git

cd rpg126541/app

```



\### Build del progetto

```bash

./gradlew build

```



\### Esecuzione

```bash

./gradlew run

```
